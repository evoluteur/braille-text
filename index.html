Braille-Text

Simple javascript and CSS to display Braille with translation on hover.

See your name in Braille

2 styles: White & Black

White Braille

Black Braille

Usage

To insert Braille in your web pages, simply use the following code.

In the page head.

  <link href="css/braille.css" rel="stylesheet" />
  <script src="js/braille.js"></script>

In the page body.

  <div id="braille"></div>
  <script>
      document.getElementById("braille").innerHTML = braille("I read Braille");
  </script>
